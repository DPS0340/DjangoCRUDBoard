# KPU C크릿쥬쥬 프로젝트 - 게시판 백엔드(Django)

## 설명

차후에 React로 프론트엔드가 개발될 게시판의 백엔드 서버입니다.

REST 방식으로 JSON을 결과값으로 .

## 팀원

박근우, 장동현, 이하늘, 이지호

## 사용 스택

개발: Python, Django

CI: Github Actions

배포: AWS Elastic Beanstalk
